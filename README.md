# **🚍 Smart Campus Transport Tracking System**

## **📖 Overview**

The **Smart Campus Transport Tracking System** is a web-based application that provides real-time tracking of campus buses and transport vehicles. It helps students and staff monitor vehicle locations, view routes, and receive transport updates, improving convenience and reducing waiting times.

## **✨ Features**

* Real-time vehicle tracking
* Route and schedule management
* Estimated Arrival Time (ETA)
* User authentication and login
* Admin dashboard for transport management
* Notifications and alerts
* Interactive map integration
* Secure data storage

## **🛠️ Technologies Used**

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js / Python / Java
* **Database:** MySQL / MongoDB
* **Maps & Tracking:** GPS and Map APIs

## **📂 System Modules**

### **👤 User Module**

* View bus locations
* Check routes and schedules
* Receive transport notifications

### **⚙️ Admin Module**

* Manage vehicles
* Update routes and schedules
* Monitor transport operations

## **🎯 Benefits**

* Reduces waiting time for passengers
* Improves transport efficiency
* Enhances campus mobility
* Provides accurate and real-time information

## **💻 Installation**

### **Prerequisites**

* Web Browser (Chrome, Firefox, Edge, etc.)
* Internet Connection
* Local Server (Optional)

### **Steps**

1. Download or clone the repository.
2. Extract the project files (if downloaded as ZIP).
3. Open the project folder.
4. Run the application using a local server or open the `index.html` file in a web browser.
5. Access the Smart Campus Transport Tracking System and start using the application.
